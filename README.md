# SocialClimateModel
The code used in the manuscript "Tipping to Climate Action: Qualitative Insights from a Social-Climate Model with a Commited Minority."

The files starting with "EnergyBalance" run the climate model (both ODE and SDE formulation) for each of the five scenarios. The file fullModel_rep.py runs the full social-climate model for a specified number of repetitions.

The Carbon_dioxide_Gt_CO2_yr file is data pulled from: Gulev, S.K. et al. (2021). “Changing State of the Climate System”. In: Climate Change 2021: The Physical Science Basis. Contribution of Working Group I to the Sixth Assessment Report of the Intergovernmental Panel on Climate Change. Ed. by V. Masson-Delmotte et al. Cambridge, United Kingdom and New York, NY, USA: Cambridge University Press, pp. 287–422. doi: 10.1017/9781009157896.004.
